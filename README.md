## Project Overview

This repository provides a compact workflow for exploring random‑walk simulation, naïve backtesting, and the evolution from GenAI‑generated code to a correct, vectorized implementation. The three notebooks illustrate the progression from a student starter quiz to the raw GenAI output and finally the refined version.

## Notebooks

### [random-walk-getting-started.ipynb](random-walk-getting-started.ipynb)
- Entry point for students  
- Mini‑quiz setup  
- Generate a random walk, create signals, run the naïve backtester  
- Designed for hands‑on practice  

### [random-walk-naive.ipynb](random-walk-naive.ipynb)
- First version produced by GenAI  
- Contains the initial naïve implementation  
- Useful for understanding mistakes, inefficiencies, and semantic drift  
- Baseline for comparison  

### [random-walk-panda.ipynb](random-walk-panda.ipynb)
- Improved version after multiple refinement cycles  
- Vectorized logic, corrected semantics, cleaner structure  
- Demonstrates how to evolve GenAI output into real, reliable code
